Make halves and more halves
ZPRAC_Lab-11_CLI-FileIO

[50 points] Write a program that manipulates the bit representation of a number and divides it by a power of 2. As input, you will be given the number as well as the power of 2 to be divided by. Note that you must use bitwise operations to obtain the final result. Do not use normal division to obtain the result. INPUT: Two integers. The first will be a positive integer in the range of 1 - INT_MAX, the second will be a number in the range of 0 - 30. OUTPUT: Output a single integer, the result of dividing the first number by 2^(second number) EXAMPLE: INPUT: 1000 3 OUTPUT: 125 Explanation: 1000 / 2^3 = 8 Note: Few marks will be reserved for good programming practices.
